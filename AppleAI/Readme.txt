
# -------------------------------------------------------
# 此规则集 ：
# -------------------------------------------------------
#
#   用于 ：对 苹果智能 apple intelligence 进行 clash 分流
#         让 Siri 调用 ChatGPT（海外AI），而不是 文心一言（国产AI）
#
#    >> 🇨🇳 国行 iOS    : 无需尝试 ！！！设备已被苹果锁死 。
#			 或 使用misaka26（请抖音搜索教程） ：https://github.com/straight-tamago/misaka26 
#
#    >> 🇨🇳 国行 MacOS  ：单纯调用本规则，无法切换到 🇺🇸 美区 苹果智能 。
#                       必须配合 enableAppleAI 脚本 和 其他设置
#
#    >> 🇨🇳 外版 苹果设备 ： 支持 ！但还需要，配合 其他设置 。
#
#    >> 总结：本规则，只适用于
#          *  "解锁后" 的 国行 🇨🇳 Mac （不含 🇨🇳 国行 iOS）
#          *  "非国行" 🇺🇸🇯🇵🇭🇰🇹🇼🇨🇦🇪🇺 海外版 iPhone、iPad、Mac



# -----------------------------------
# 针对 🇨🇳 国行 Mac OS ：
# -----------------------------------
#
#  需解除 3把锁，才能使用上 🇺🇸 美国的苹果智能 （ChatGPT for Siri）
#    1. 设备锁 （ 仅针对 中国大陆国行 ）
#    2. 位置锁 （ GPS位置 位于 🇨🇳 中国大陆 ，则无法使用 ChatGPT for Siri ）
#    3. IP锁 （ IP归属地 位于 🇨🇳 中国大陆 ，则无法使用 ChatGPT for Siri ）
#
#   第1把锁 ：通过 enableAppleAI 脚本 可以解除 （仅支持 解除国行Mac，而非iOS）
#            （ 移步：https://github.com/kanshurichard/enableAppleAI ）
#   第2把锁 ：通过 系统设置，可以解除 
#   第3把锁 ：通过 分流规则，可以解除，


# -----------------------------------
#   针对 非国行 iPhone、iPad、Mac ：
# -----------------------------------
#
#   ⚠️⚠️ 无设备锁。只需要解除， 第2、3把锁。



################################################################
#  如何解除 设备锁 ？
################################################################
#
#  只有 🇨🇳国行Mac 能解除此锁。而 🇨🇳国行ipad、🇨🇳国行iphone，都无法解除！
#  如何解除：
#  - 先完成以下设置，然后，再 Mac上 运行 enableAppleAI 脚本！（注意先后顺序）
#    （ 脚本下载地址：https://github.com/kanshurichard/enableAppleAI ）
#
#  1.  系统账号：必须登录 非中国区 的 苹果账号（不能不登录❕❕） 
#        外国账号 = iCloud 账号 + AppStore 账号 （两者 缺一不可⚠️） 
#  2.  系统地区：必须选择 非中国（如 美国） 
#        设置 -> 通用 -> 语言与地区 -> 地区 ->  美国
#
#  完成上述两步设置后，运行 按照enableAppleAI的规范，去运行脚本。才能解除设备锁。
#
#  注意： 
#    > A. 运行脚本前，必须关闭SIP，运行后可重新开启
#    > B。每次大版本升级（如v26 -> v27 ），未来可能需要重新跑一遍本脚本（目前不需要）
#    > C。🇺🇸🇯🇵🇭🇰🇹🇼🇨🇦🇪🇺 非国行 mac、iPhone、iPad ：无需 本段落内 任何步骤



################################################################
# 如何解除IP锁 ？（或称 如何让 Siri 绑定 ChatGPT账号 ？）
################################################################
#
#  必须是海外IP，才能解除IP锁，即，才能让 Siri绑定ChatGPT。
#
# 	3. 【目标：解除 IP锁】 应用 “本分流规则 + chatgpt分流规则” 后，即可绑定
#
#
#  ⚠️⚠️⚠️ 要特别注意 ⚠️⚠️⚠️ 
#        > 能够绑定ChatGPT账号的前提，是你能在 " 浏览器 + ChatGPT APP " 中，
#          正常访问和登录ChatGPT ！如果不能正常访问ChatGPT官网，那说明
#          你的节点IP有问题，或者你的VPN代理设置不对

 

################################################################
# 如何解除 位置锁 ？（或称 如何让 Siri 调用 ChatGPT ？）
################################################################
#
#  必须关闭Siri的地理位置获取，才能解除位置锁（即，LBS锁）。
#  也就是说，不能让Siri，检测到你GPS位置在中国，不然，你即便完成之前的所有步骤，
#  苹果设备也依然不会调用海外AI（ChatGPT），哪怕你不是国行设备。
#
#  2. 将Siri 的 地理位置禁用：
#        设置 -> 隐私与安全性 -> 定位服务 -> Siri （关闭）
#
#
# ⚠️⚠️⚠️ 要特别注意 ⚠️⚠️⚠️  
#      > 即便是非国行机，如果你GPS显示你在中国大陆，哪怕你已使用了本分流规则，
#        如果不进行这一步操作，那么Siri仍然会调用“文心一言”，而不是ChatGPT。
#      > 完成上述操作后，需要手动在Siri对话框输入，请调用ChatGPT。系统才会
#        调用ChatGPT。



################################################################
# 规则已经被命中，但是连不上 苹果智能（无法让Siri调用ChatGPT、无法在siri中登录Chatgpt） 怎么办 ？
################################################################
#
#   >  检查点 1 : 在 浏览器 APP 、 ChatGPT APP 中，你能不能正常 登录、访问 ChatGPT？
#                * ❌ 如果这一步都不能满足，那请你更换 代理节点（ChatGPT会封杀流氓代理节点）
#	         # ❌ 注意：如果在 苹果智能中登陆 C哈他GPT账号，你必须添加ChatGPT的黑名单分流，你确定你添加了吗？
#
#
#   >  检查点 2 : 苹果智能与ChatGPT不同。其大量使用了Quic。
#                所以你还要检查：
#                *  你的VPN，其连接，协议支持不支持Quic ？ 如果不支持，则无法连接。
#                *  你的VPN，是否使用了链式代理 ？ 因为有些链式代理，是无法支持Quic协议的。
#
#
#   > 检查点 3 ： 注意，以下苹果规则之间是有交叉的，请检查你的其他（更靠前解析的）苹果规则
#		 * Siri/Spotlight/Lookup/Safari/News/信息/音乐
#
#
#   >  实际上Grok AI也有类似问题。如：在不支持IPv6的网络上，可能会数分钟都连接不上Grok 




################################################################
# 以上检查点都排除了，还是连不上， 怎么办 ？
################################################################
#
#  ⚠️ Siri 在调用 苹果智能 时，会优先使用IPV6解析。也就是说，如果你的翻墙规则中，允许域名
#	在DNS解析时，生成IPV6结果。但是你的VPS节点 或 Stash配置又关闭了IPV6路由功能
#	那么，Siri将无法调用ChatGPT
#
#	方法 1 : Stash 启用 IPV6 后可解决此问题 ：
#
#		 - 开启路径为： Stash -> 设置 -> 网络设置 -> 启用 Tunnel IPV6 路由
#
# 		 - ⚠️ 惩罚：在Stash上，开启IPV6支持后，会导致 手机在局域网内 观看 
#			Apple Homekit摄像头 时，无法观看实时视频（手机在公网时 则不受影响）。
#			且在Stash上，无论怎么设置 局域网直连规则，都无法解此问题。在Clash Meta
#			中，可以通过 内网分流规则 从而绕开此问题。
#
#
#	方法 2 : 在DNS配置中关闭IPV6解析，如下：
# 		dns                     :
#    		enable                  : true         # 如果配置为关闭，则将仅仅使用系统 DNS    : true \ false
#    		ipv6                    : false        # 设置：关闭 基于IPV6的DNS查询 ✅✅✅✅✅ （这里是要更改的地方！！！！）  #
#
#
#
# ⚠️ 注意：方法2，无法使用在Stash 3.x中，因为这个APP不支持手动关闭IPV6解析。已经向开发者反馈多次，压根没人回复，这是服了。希望其他人看见了，向开发者反馈。 在2025年，我反馈了N次BUG向他们，居然都不回复。也不知道在想什么。
#
#
# ❌❌❌❌❌ 要特别注意，IPv6问题，不仅仅会导致Siri无法调用C哈他GPT，也会导致用户无法在设置siri的界面登录中登录ChatGPT。系统会显示：“无法登录，登录账户时出现问题” ，哪怕你已经输入完了登录密码和验证码，依旧无法登录。
#

# 
# 三组的后缀，分别为：
# 
#   + 后缀：No_Resolve	（第一组）
# 
#   + 后缀：无		（第二组）
# 
#   + 后缀：Doamin		（第三组）
#   + 后缀：IP
# 
# 以上三组，三选一，优先选择最后一组（Domain + IP），在移动端（Stash for iOS），这种写法极大增加匹配速度和减少内存占用。
