# 用途：根据目的网站的IP，选取最近国家的VPS代理节点进行分流（通过GeoIP） 
#
# 使用建议：
#
# 	1. 不建议使用_No_Resolve版本		（不做域名解析，则无法充分分流）
# 	2. 安排在Final兜底规则前，做最后的分流	（以避免影响到其他网站的专属分流控制）

#   4, 对于白名单分流来说，不可替代Final （有些IP并没有包括在GeoIP当中，还需要Final做最后的兜底）
